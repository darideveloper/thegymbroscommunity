# Proposal: Add Pinyon Script and Lato fonts

## Why
The current typography does not align with the brand identity. Adding Pinyon Script and Lato will provide a more polished, professional appearance.

## What Changes
- Update `package.json` to include Fontsource packages.
- Import fonts in `global.css`.
- Update global styles to apply these fonts.

## Goal
The application SHALL install and configure Pinyon Script (for titles) and Lato (for body text) using Fontsource to improve typography. The application MUST convert the "The Gym Bros Community" text to an `h1` heading using the Pinyon Script font.

## Scope
- The application SHALL update `package.json` to include Fontsource packages.
- The application SHALL import fonts in `global.css` or the main layout.
- The application MUST update global styles to apply these fonts.
